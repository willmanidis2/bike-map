How to export your emails

Bulk export all your ride receipts at once using Google Takeout:
  1. In Gmail, search: from:no-reply@updates.citibikenyc.com subject:"Ride Receipt"
  2. Select all results, click "Labels" (tag icon) and create a new label called "Rides"
  3. Go to takeout.google.com — click "Deselect all", then scroll down and check only Mail
  4. Click "All Mail data included" → select only the "Rides" label
  5. Click "Create export", wait for the ZIP, download and unzip it — you'll find an .mbox file
  6. Upload that .mbox file here
Export all your ride receipts in one drag:
  1. Open Outlook Desktop and search: "Ride Receipt" from your bikeshare
  2. Press Ctrl+A (or Cmd+A on Mac) to select all results
  3. Drag them to a folder on your Desktop — each becomes a .eml file
  4. Upload the whole folder here (or select all the .eml files)
Outlook Web? Unfortunately there's no bulk download. We recommend adding your account to Outlook Desktop or Apple Mail for the drag-and-drop export.
